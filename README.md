# Agentic Economy on Arc - Hackathon Project

## Overview

This project is built for the **Agentic Economy on Arc** hackathon (April 20-26, 2026), powered by Circle and Arc. The mission is to build the next generation of agentic economic applications using Circle's Nanopayments infrastructure with settlement on Arc in USDC.

**Key Innovation**: For the first time, high-frequency, usage-based transactions between users, APIs, and AI agents can operate without gas overhead eroding margin. Arc provides stablecoin-native settlement with predictable, dollar-denominated USDC fees and deterministic, sub-second finality.

## Challenge Tracks

Choose one track to focus your project:

### 🪙 Per-API Monetization Engine
Build an API that charges per request using USDC, demonstrating viable per-call pricing at high frequency.

### 🤖 Agent-to-Agent Payment Loop
Create autonomous agents that pay and receive value in real time, proving machine-to-machine commerce without batching or custodial control.

### 🧮 Usage-Based Compute Billing
Design an application that charges per query, per compute unit, or per task with real-time settlement aligned to actual usage.

### 🛒 Real-Time Micro-Commerce Flow
Build a buyer or seller experience where economic activity is triggered and settled per interaction, not per subscription.

## Technology Stack

### Required
- **Arc**: EVM-compatible Layer-1 blockchain for settlement
- **USDC**: Native gas token and stablecoin on Arc
- **Circle Nanopayments**: Infrastructure for economically viable sub-cent, high-frequency transactions

### Recommended
- **Circle Wallets**: Secure, programmable wallets for users and agents
- **Circle Gateway**: Unified USDC balance across chains
- **Circle Bridge Kit**: Cross-chain USDC movement
- **x402 Payment Standard**: Web-native payment verification and submission

### AI & Models
- **Google Gemini 3 Flash**: Optimized for transactional agents and real-time commerce
- **Google Gemini 3 Pro**: Advanced reasoning for complex financial workflows
- **Featherless AI**: Specialized AI models for coding, medical, legal, and multilingual use cases

### Optional Tools
- **Vyper**: Smart contract language with agentic payment tooling
- **Circle-titanoboa-sdk**: Vyper + x402 payment integration
- **ERC-8004-vyper**: Trust layer for autonomous agents
